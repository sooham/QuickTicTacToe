#Quick Tic-Tac-Toe

A graphical Tic-Tac-Toe client activated from the shell.
Uses Minimax artificial intelligence algorithm to replicate a human player.

![an image of the program](./image.tiff "Logo Title Text 1")

### Dependencies
* Python 3
* Tkinter
